# Análisis Estadístico de Enfermedad Cardíaca

## Objetivo

Realizar un análisis estadístico de un conjunto de datos de salud utilizando R, aplicando limpieza de datos, análisis exploratorio y un modelo de regresión logística.

## Estructura

- `data/`: datos originales y procesados.
- `scripts/`: scripts del proyecto.
- `outputs/`: tablas y gráficos generados.
- `docs/`: documentación.

## Ejecución

Ejecutar los scripts en el siguiente orden:

1. `00_config.R`
2. `01_limpieza.R`
3. `02_exploracion.R`
4. `03_modelo_regresion.R`
